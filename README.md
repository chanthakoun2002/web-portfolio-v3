# Andrew Chanthakoun – Developer Portfolio
Work in progress
Version 3 of my portfolio

This repository contains the source code for my personal developer portfolio website.

The portfolio showcases my projects, skills, education, and ways to contact me. It was built to demonstrate my experience with modern frontend development and to provide a central place to view my work.

## Technologies Used

- Angular
- TypeScript
- HTML
- CSS
- Angular Material

## Features

- Responsive modern UI
- About section with education background
- Skills section
- Projects showcase
- Contact section with links to email, LinkedIn, and GitHub
- Clean dark-themed design
